# core-ai

AI layer package for Internal Sea.

This package will contain future capabilities such as:

- Context builder (assemble product, work and team context for prompts)
- Summarization of risks, decisions and activity
- Assistant orchestration hooks used by the API

No AI features are implemented in the foundation phase. Structure data and APIs so context can be added without breaking changes.

See Phase 10 in [docs/IMPLEMENTATION_ROADMAP.md](../../docs/IMPLEMENTATION_ROADMAP.md).
