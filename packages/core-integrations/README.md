# core-integrations

External integrations package for Internal Sea.

This package will contain connectors and sync logic for tools such as:

- Jira
- GitHub
- Microsoft Teams
- Slack
- BI platforms and data platforms

Integrations should be optional and isolated so the core product works without them.

Implementation is planned for post-MVP phases. See [docs/IMPLEMENTATION_ROADMAP.md](../../docs/IMPLEMENTATION_ROADMAP.md).
