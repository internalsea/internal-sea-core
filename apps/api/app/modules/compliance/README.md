# Compliance module

Policies, rules, controls, compliance checks and evidence for Internal Sea.

## MVP scope

- Manual compliance checks for data products, projects, internal projects, teams and capabilities
- Evidence links to `FileAsset` records from the Files module
- Optional automatic `FileAttachment` when evidence is added (supported subject types only)
- No automated rule execution, schedules, approvals or auth yet

## API

Base path: `/api/v1/compliance`

See OpenAPI docs at `/docs` for full endpoint list.
